📖 PANDUAN PENGGUNAAN — Clinical Pathway Maker

⚡ Cara Cepat Mulai

  1. Isi informasi rumah sakit — Nama RS, nama institusi (universitas), Nomor CP, Tanggal Berlaku, Nomor Revisi.
  2. Isi data pasien — Nama, Jenis Kelamin, Tanggal Lahir, No. Rekam Medik, tanggal masuk/keluar, lama rawat, ruang rawat, dll.
  3. Pilih Spesialisasi dari dropdown, lalu ketik nama diagnosis di kolom Judul Diagnosis / Tindakan.
  4. Klik ⚡ Proses — seluruh konten CP (15 seksi, hari rawat, label hari) terisi otomatis sesuai diagnosis.
  5. Edit baris — klik langsung pada teks di tabel untuk mengedit. Klik + untuk tambah baris, 🗑 untuk hapus.
  6. Klik sel hari (kolom berwarna) untuk ubah status: Wajib → Boleh → Tidak Berlaku → Wajib.
  7. Isi nama & NIP pejabat (Komite Medik, Direktur) di panel setup — muncul otomatis di blok tanda tangan.
  8. Export — klik ⬇ Export Word (.doc) untuk unduh, atau 🖨 Print untuk cetak langsung.
💡 Jika diagnosis tidak terdeteksi, sistem akan menampilkan pilihan manual. Gunakan filter Spesialisasi agar pencarian lebih akurat.

⚙️ Tips & Catatan

  • Jumlah hari rawat dapat diubah (1–30) di field "Hari Rawat" — isi baris menyesuaikan otomatis.
  • Label kolom hari dapat dikustomisasi bebas (contoh: Pre-Op, Operasi, Post-Op 1, Post-Op 2).
  • Baris bertanda ⚠️ = bukti klinis masih lemah — tampil di preview tapi tidak ikut ter-export ke Word.
  • Tombol ↺ Reset menghapus semua isi CP dan kembali ke kondisi kosong.
  • Tekan Esc untuk menutup jendela ini.
  • Aplikasi ini berjalan 100% offline — tidak memerlukan internet, server, atau instalasi tambahan.
